Question-1: What are the different ways to select an element in the DOM?

getElementById(), getElementByClassName(), getElementByTagName(),querySelector(),querySelectorAll().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML - returns all contents including tags, innerText - only returns the original content considering style, textContent - only returns the original content without considering style.

Question-3: What is event delegation in the DOM?

When parent handles events for chlid using event bubbling, it is called event delegation.

Question-4: What is event bubbling in the DOM?

When an event starts on a child element and moves up to its parent and beyond it's called event bubbling.

Question-5: How do you create, add, and remove elements using JavaScript?

I can create element using document.createElement(), add element using appendChild() or insertBefore() and remove element using removeChild() or remove().
